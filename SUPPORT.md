# Support

For setup and usage questions:

1. Check `README.md`, especially **Quick start**, **Auto-detection**, and
   **Troubleshooting**.
2. Run:

   ```bash
   python3 config.py --detect-runtime
   python3 llama_server_manager.py status
   ```

3. Search existing GitHub issues.
4. Open a new issue using the bug report or question template.

Please include OS, Python version, `llama.cpp` build type, and relevant logs.
Do not include API keys, private prompts, or sensitive paths.
