# Code of Conduct

## Our Pledge

We want `llama.cpp Control Deck` to be a welcoming project for people building
local AI tooling. Be respectful, practical, and kind.

## Expected Behavior

- Use welcoming and inclusive language.
- Assume good intent when something is unclear.
- Keep criticism focused on code, docs, behavior, or reproducible facts.
- Respect different levels of experience.
- Avoid publishing private information, API keys, logs with sensitive prompts,
  or production host details.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Personal attacks or sustained hostile behavior.
- Posting private or sensitive information without consent.
- Spam, trolling, or unrelated promotion.

## Enforcement

Maintainers may edit, hide, or remove comments and issues that violate this
code of conduct. Repeated or severe violations may result in a ban from project
spaces.

If you need to report a problem, open a private security advisory when the issue
is sensitive, or contact the maintainers through the repository owner profile.
